# Shader Editor Pie Menu

A Blender addon that adds a pie menu (Tab key) in the Shader Editor with quick shortcuts:

- **SD Texture Import** - Import Substance Designer PBR textures and auto-connect them to a Principled BSDF node (Base Color, Normal, Roughness, AO, Metallic, Opacity).
- **Mix Color** - Create a Mix Color node and connect it to selected nodes.
